## Installing:

1. Clone the repo
```
git clone https://github.com/DanielStochel/Todo-app
```

2. Directory 
```
CD ( folder that You have created)
```

3. In terminal Install packages
```
npm install 
```

5. Enjoy the application !
```
npm start
```
